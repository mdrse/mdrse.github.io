I plan on first writing “user stories”, a user story is a feature that a user might want for example: “As a user I would like to have a contact form so I can fill in my name and email address and make a comment or ask a question.”

See: https://stormotion.io/blog/how-to-write-a-good-user-story-with-examples-templates/

Once the “user stories” are written I plan to use those to write a specification for the website using a plan similar to the one available here: https://wemakewebsites.com/blog/how-to-write-a-website-specification/

I will then wireframe the design using inVision.

Once the spec & wireframe is ready I will transfer the specification into Trello cards which will be tasks to produce the specification.

I plan on using Github for backup archiving and version control in conjunction with WordPress to produce and publish the website.

I will call upon the other members of the Code Nation course for testing and suggestions improvements.

Any documentation required will be in Google Docs.

As well as the submitted HTML the final product will be available to view at https://david-r-southall.me/
