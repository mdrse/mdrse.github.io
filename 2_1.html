Review the ongoing use of IT tools and change approach as needed – in making your own website, are you happy with the tools you used? E.g. Visual Studio Code, Trello

I used a range of tools to complete the task and judged they were adequate to complete the task. I started with Google Docs: https://www.google.co.uk/docs/about/ ; this allows a user to write a document which autosaves as you create and also allows you to share as a link either editable by team members or just for viewing. Extremely useful for collaborating. Trello: https://trello.com/ is another useful collaboration tool allowing the user to define tasks as cards and assign them, however, Trello is also extremely good when working alone as a personal planner and task monitor.

I did change my mind about inVision: https://www.invisionapp.com/ ; inVision is essentially a electronic whiteboard but I found it too freehand for my tastes so I switched to draw io: https://www.draw.io/ draw io allows you to create a wireframe by dragging shapes onto the workspace and connecting with arrows so was much more suited to my needs. Visual Studio Code was fine for HTML and CSS: https://code.visualstudio.com/ ; Visual Studio Code also nicely integrates with https://github.com/ where I created a repository for backup: https://github.com/mdrse/codenationhtmldrs/tree/master/codenationhtmldrsday010 I also added a bell & whistle by creating a short video in https://www.doodly.com/
